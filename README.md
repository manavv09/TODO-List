# ✅ Todo List App

A clean and modern Todo List app made using **React** + **Context API** with **LocalStorage** support.

## 🌐 Live Demo
https://manavv09.github.io/TODO-List/

## 🚀 Features
- Add Todo
- Edit Todo
- Delete Todo
- Mark as Completed
- LocalStorage Support
- Clean UI (Tailwind CSS)
